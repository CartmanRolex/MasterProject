\section{Results}
\label{sec:experiments}

This section reports the experimental evidence collected so far. The first result establishes a monolithic full-episode baseline. The later results characterise the subtask model and identify the formal metrics that remain to be collected once orchestrated evaluation completes.

\subsection{Full-Episode Imitation Baseline}

The first experiment evaluates whether changing the policy class is sufficient to improve the complete pick-and-place task when the task is treated as one monolithic episode. ACT and SmolVLA are trained on the same public HuggingFace dataset of 60 full-task demonstrations. In this setting, SmolVLA receives a single instruction for the whole episode, so the experiment does not evaluate meaningful language-conditioned control. Both policies are asked to solve the full task without any subtask-level monitoring.

ACT and SmolVLA achieve comparable performance. This confirms that, under identical training conditions and with a single fixed instruction, the VLA model choice does not determine outcome. Selecting SmolVLA is therefore not a claim about its superiority as a policy class, but a choice motivated by its language conditioning capability --- which this experiment does not exercise and the subsequent subtask formulation is designed to exploit.

\subsection{Manual Subtask Dataset}

The dataset contains 921 successful subtask episodes, each annotated with a relative-position instruction and collected in a randomised pick order. The resulting SmolVLA model qualitatively follows target-specific grasp instructions — the enabling capability for the orchestrator: after a subtask failure, the system can issue a corrective instruction rather than aborting the full episode.

\subsection{Autonomous Data Generation}

Autonomous data generation is currently in progress. The orchestrated execution system is being used to collect successful subtask trajectories while filtering out failed subtask attempts. This is expected to produce a dataset that better reflects states encountered during autonomous execution than the manually teleoperated dataset alone.
